# gulp-cloudflare
> cloudflare plugin for [gulp](https://github.com/wearefractal/gulp)

## Usage

First, install `gulp-cloudflare` as a development dependency:

```shell
npm install --save-dev gulp-cloudflare
```

Then, add it to your `gulpfile.js`:

```javascript
var cloudflare = require("gulp-cloudflare");

gulp.task("purgeCache", function(callback) {
    var options = {
        token: cloudflareAuthKey,
        email: cloudflareEmail,
        zone: cloudflareZone
    };
    cloudflare(options);
    callback();
});
```

## API

### cloudflare(options)

#### options.token
Type: `String`  
cloudflare Auth-Key
Required: true
#### options.email
Type: `String`  
Required: true
#### options.zone
Type: `String`  
Required: true
#### options.skip
Type: `Boolean`  
Required: false
